PM 1: Requirements Specification / Project Scope Statement

The project entails creating a web based database system to keep track of a collection of audio compact disks. The data will be held on a "server" which will be accessible to users connected to the Internet.

The database will contain data referring to Audio compact discs, which will include the following data:

  • Album Title
  • Artist
  • Album Classification
  • Audio Clip
  • Label
  • # of Tracks
  • Track Titles
  • Track Times
  • URLs
The data will be available for viewing in HTML. The project will be implemented using the following tools:
  • CGI - will be used for transferring information between the World Wide Web server and the CGI programs.
  • Perl - will be used for writing the CGI programs.
  • MSQL - (a public domain UNIX based database server) will be used for storing and retrieving the data.
The implementation of the project will require the interfacing of the above mentioned tools. The goal is to allow the user of the software the following activities within the CD database:
  • Search the database for existing CD's
  • Add new CD's to the database
  • Edit existing CD information
  • Delete any existing CD inforamtion
  • Allow the user to technically download audio samples from the albums. However, since this brings issues of copyright into play, there will be NO samples actually available for the user to download.
  • Allow the user to look at URLs (if they exist) related to the album or artist in question.

The project will not allow the user any of the following activities:

  • Alter the structure of the database.
  • The system will not organize the CDs in any way other than is offered.
  • It will not allow searches, other than those explicitly offered.
  • It will not facilitate purchasing of CDs, nor will it forward the user to someplace where they can.
