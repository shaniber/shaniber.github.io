Welcome to my website! It was last updated 19 Sept 2000.

Here, I shall chronicle my attempts at brushing up on my programming skills, and learning the Allegro Game Programming Library. I'll keep my last 5 projects on the front page for easy viewing. Yes, I realize that it isn't actually shaniber.net, but someday, baby, someday!

My current setup at home is a cobbled together AMD K6-233 with 64MB of RAM and a 4GB of hard drive. I'm running Windows 98. I'm using Microsoft Visual Studio 6 Pro, the latest DJPGG from Delorie Software, and the 3.9.32 WIP of Allegro. Right now, I'm just posting the source code for my projects, but I'm hoping to work them up into a sort of tutorial for people who are just starting out.
