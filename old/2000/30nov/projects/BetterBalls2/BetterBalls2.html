This is the Better Balls 2 project. It is a significant enough improvement over Better Balls that I think it deserves to have a page of its own!

I improved the physics of this one, and the balls don't disappear as much. (Actually, I lifted the physics from the solution for a graphics project I did in University, but I don't think that's bad, is it?).

Click the screenshot for a closer look.

I used George Foot's Allegro Vivace Tutorial again here, since this is just an extention of Better Balls.

There are 15 circles this time, once again 'randomly' generated. I also added a little code to make sure that any circles that appear off of the edge of the screen (as some seem to do!), are forced back inside the screen.

Problems with the current version: the circles still get stuck together, but it looks kinda cool, so i decided that this is a feature. Deal. :)
